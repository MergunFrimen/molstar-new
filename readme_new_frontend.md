## Building & Running new frontend

### Preparation:
After cloning the repository, switch to `cvsx-new-query-app` branch

### Build:
    npm install
    npm run watch-viewer

### Serve:
In another terminal

    npm run serve

Then open http://127.0.0.1:1338/build/viewer/index.html
